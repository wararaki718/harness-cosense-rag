# PRODUCT_SENSE.md

Product philosophy and sense for `harness-cosense-rag`.

## Philosophy
The goal of this RAG system is to unlock the collective knowledge stored in Cosense projects. By combining powerful sparse vector search (SPLADE) with state-of-the-art LLMs (Gemma3), we aim to provide a "second brain" that can accurately answer questions based on personal or organizational notes.

## Core Value
- **Accuracy**: Using SPLADE for precise retrieval.
- **Privacy/Local-First**: Integrating with Ollama for local LLM execution.
- **Integration**: Seamlessly pulling from Cosense as a primary source of truth.
